# BugOSC
BugOSC， 是一款微信小程序版的 OSC 控制器，可向局域网内的设备发送标准 OSC 数据。
欢迎在这个repo里提功能建议、反馈Bug，请提交 [Issue](https://github.com/avantcontra/bugosc/issues) 。
